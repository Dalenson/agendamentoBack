
# Serviço de autenticação utilizando JWT juntamente com API para um sistema de cadastramento de agendamentos
Serviço desenvolvido para praticar o desenvolvimento de aplicação utilizando JWT e API para cadastramento de agendamento.
## Stacks utilizada

**Back-end:** Spring Boot, MySql, Heroku, JWT.
